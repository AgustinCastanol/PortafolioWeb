## ADDED Requirements

### Requirement: Display Fruit Map Project
The system MUST display the Fruit Map project in the portfolio list with its designated thumbnail (image 8).

#### Scenario: User views the portfolio projects
- **WHEN** the user navigates to the portfolio projects page
- **THEN** they see the Fruit Map project card
- **AND** the project card uses image 8 as its thumbnail

### Requirement: Stardew Valley Aesthetic
The system MUST apply a specific Stardew Valley visual theme to the Fruit Map project representation.

#### Scenario: User inspects the Fruit Map project details
- **WHEN** the user views the Fruit Map project card or details
- **THEN** the UI elements (borders, fonts, colors) reflect the Stardew Valley aesthetic
- **AND** other projects are not affected by this aesthetic

### Requirement: View Transitions Navigation
The system MUST utilize View Transitions when navigating to or from the Fruit Map project details.

#### Scenario: User clicks on the Fruit Map project
- **WHEN** the user clicks on the Fruit Map project card
- **THEN** a smooth view transition animation occurs into the project details view
